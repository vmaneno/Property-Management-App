Features & Capabilities

Everything you need to run a professional property management business
Page 3 of 4
Core Management Tools
Property Management Register and track multiple properties with client linkage, manager details, utilities and status — with collapsible group views.
Unit Management Track every unit's rent, deposit, occupancy and tenant assignment. Filter by status, search by property or unit number.
Tenant Management Complete tenant profiles — ID, contacts, lease dates, rent, deposit and payment history. Grouped and searchable by property.
Transaction Ledger Post credits and debits with running balances, references, verification status and optional document attachments.
Bulk Excel Import Upload units and tenants in bulk using downloadable Excel templates with built-in row-level validation and error reporting.
Live Dashboard At-a-glance KPIs — total portfolio size, occupancy rates, revenue summaries and recent transaction activity.

Security & Access Control
JWT Authentication Every session is secured with a signed JSON Web Token — no session data stored in the browser.
Role-Based Access Master, agency, client and tenant users each see only the data relevant to their role — nothing more.
Password Reset Secure email-based password reset flow for all user roles with time-limited reset tokens.
Cloud Database All data stored in Supabase (PostgreSQL) — reliable, scalable and accessible from anywhere.
