Section 1 — System Overview & Data Hierarchy

Understanding the full scope of Master Admin access

The Master Admin has unrestricted access to every record in the system. All data is organised in a strict five-level hierarchy:

🛡️ Master Admin (full system access — you)
→ 🏙️ Agencies (property management companies)
→ 👔 Clients (property owners managed by an agency)
→ 🏢 Properties (buildings/estates owned by a client)
→ 🚪 Units (individual rentable spaces within a property)
→ 👤 Tenants (occupants of units)
→ 💳 Transactions (payments and charges per tenant)

Master Admin Responsibilities

✅ What Master Admin Can Do

  • Create, edit, and delete agencies
  • Create, edit, and delete clients under any agency
  • Manage all properties, units, and tenants system-wide
  • Post, edit, and delete transactions for any tenant
  • Verify (authenticate) transactions
  • Generate and export reports across the whole system
  • Reset the entire system (deletes all data)

⚠️ Restrictions & Rules

  • Cannot delete a record with dependent children (e.g. agency with clients)
  • Cannot delete verified transactions
  • Cannot delete active clients, tenants with transactions, or occupied units
  • System Reset is permanent and irreversible

Section 2 — Logging In & First-Time Setup

Accessing the Master Admin portal

Logging In

  1. Navigate to the Property Management App URL in your browser.

  2. Click the Master Admin card on the landing page.

  3. Enter your Username and Password. Use the 👁 icon to reveal the password if needed.

  4. Click "Sign In" (or press Enter). You will land on the Master Admin Dashboard.

First-Time Setup

Before you begin: You need your Master Agency Code and the telephone number registered in the system.

  1. Click "First Time Setup →" on the Master login card.

  2. Fill in all fields:

FieldWhat to EnterRequired
Agency CodeThe master agency code for this installationYes
TelephoneTelephone number registered with the systemYes
New UsernameChoose your admin usernameYes
New PasswordMinimum 6 characters — use a strong passwordYes
Confirm PasswordRe-enter your password exactlyYes

  1. Click "Set Credentials" → "← Back to Login" → sign in.

Security: Store your Master Admin credentials in a secure password manager. Access to this account grants full control over all system data.
🔓

Section 3 — Resetting a Forgotten Password

Recovering Master Admin access

  1. On the Master login card, click "Forgot Password? →".

  2. Enter your Username and click "Send Code". Your email application opens with a pre-filled reset code email.

  3. Send the email. The form advances to Phase 2.

  4. Enter the 6-digit reset code from your email.

  5. Enter your New Password and Confirm Password.

  6. Click "Reset Password". Sign in with the new password.

Important: Reset codes expire in 15 minutes. Request a new code if yours expires.

  2. The table shows: Code, Name, Contact Person, Email, Telephone, City/Country, Currency, Active, and Actions.

  1. Click "Add Agency" (top right).

FieldDescriptionRequired

EmailAgency primary email addressYes

City / CountryAgency headquarters locationOptional

Section 8 — Managing Units

Add, edit, and bulk upload units system-wide

Viewing Units

Click "Units". Units are grouped by property. Use the search bar and status filter (All / Occupied / Vacant) to find specific units.

Adding a Single Unit

  1. Click "Add Unit" (top right).

  2. Select the Property, then fill in: Unit Number, Rent Amount, Deposit Amount, Bedrooms, Bathrooms, Floor, Status (defaults to Vacant), and Notes.

  3. Click "Save Unit".

Bulk Uploading Units (Excel / CSV)

  1. Click "Units Upload" → click "Unit Upload Template" to download the Excel template.

  2. Open the template and fill in one unit per row. Required columns:

ColumnFormat / NotesRequired
PropertyCodeMust match an existing property code (e.g. PR001)Yes
UnitNumbere.g. 101, A2 — must be unique within the propertyYes
RentAmountNumeric (e.g. 1200.00)Yes
DepositAmountNumeric (e.g. 2400.00)Yes
BedroomsInteger (e.g. 2)Yes
BathroomsInteger (e.g. 1)Yes
FloorInteger (e.g. 3)Yes
UnitStatusVacant or OccupiedYes

  1. Save your file and upload it in the dialog. Review the validation preview.

  2. Click "Import X Unit(s)" to import all valid rows.

Editing & Deleting Units

Click ✏ to edit a unit's details. Click 🗑 to delete — blocked if the unit is Occupied or has associated tenants/transactions.

Section 10 — Managing Transactions

Full transaction lifecycle management across all agencies

Viewing Transactions

Click "Transactions". The table shows all transactions system-wide with: Date, Agency, Client, Tenant, Property, Description, Amount, Reference, Status, and Actions. Use the Agency filter to narrow the view.

Posting a Single Transaction

  1. Click "Post Transaction" (top right).

  2. Select Agency → Client → Property → Tenant (each dropdown cascades based on the previous selection).

  3. Enter: Date, Amount (positive = credit/payment, negative = debit/charge), Description.

  4. Add an optional Note and upload a supporting Document (required — PDF, JPG, PNG, DOC, DOCX).

  5. Click "Post". A unique transaction reference number is auto-generated.

Bulk Posting Transactions

  1. Click "Bulk Posting" (top right).

  2. Select Agency → Client → Property (cascading dropdowns). Active tenants in the selected property appear as checkboxes.

  3. Choose Charge Type: "Monthly Rent" uses each tenant's individual rent; "Custom Charge" applies a fixed amount to all selected tenants.

  4. Enter a Description and optional Note.

  5. Check/uncheck tenants. Use "Select All" or "None" for quick selection. A count shows how many are selected.

  6. Click "Bulk Post". One transaction is posted per selected tenant.

Editing a Transaction

Click ✏ on any transaction → update Date, Amount, Description, Note, or replace the attached document → click "Save Changes".

Deleting a Transaction

Click 🗑. Only Pending transactions can be deleted. Verified transactions are permanently locked.

Transaction Status: Transactions post with status Pending. Once verified, they show as Verified and can no longer be edited or deleted. Verification is managed by the Master Admin or Agency.

Section 11 — Print Reports

Generate, print, and export system-wide reports

Accessing Reports: Click "Print Reports" in the sidebar. Four report types are available.

Report 1: Property Transactions

  1. Select a Property from the dropdown.

  2. Enter a Date From and Date To range.

  3. Click "Generate". The report shows all transactions in that date range with Debit, Credit, and running Balance columns, plus a Closing Balance at the bottom.

  4. Click Print, Export Excel, or Export CSV.

Report 2: Tenant Statements

  1. Optionally select a Property to filter the tenant list.

  2. The tenant list appears with checkboxes — all are ticked by default.

  3. Tick/untick tenants as needed, then click "Generate".

  4. One full statement is produced per selected tenant, each with transaction history and closing balance.

  5. Click Print (each statement prints on its own page), or export to Excel/CSV.

Report 3: List of Tenants

  1. Optionally filter by Property and Status (All, Active, Inactive).

  2. Click "Generate" to produce a full tenant register. Export or print as needed.

Report 4: Units List

  1. Optionally filter by Property and Status (All, Occupied, Vacant).

  2. Click "Generate". The report lists all matching units with rent, deposit, and occupancy status.

Export formats: All reports can be exported as Excel (.xlsx) or CSV. Excel exports can be opened in Microsoft Excel, Google Sheets, or similar applications for further analysis.

Section 12 — Profile, System Reset & Sign Out

Account management and critical system administration

Updating Your Profile

  1. Click "My Profile" in the sidebar.

  2. Update your Email, Telephone, or Address as needed.

  3. To change your password, fill in the New Password and Confirm New Password fields (minimum 6 characters).

  4. Click "Save Changes".

Password security: Use a strong, unique password for the Master Admin account. Change it immediately if you suspect it has been compromised.

System Reset — DANGER ZONE

⚠️ WARNING — IRREVERSIBLE ACTION The System Reset permanently and immediately deletes ALL data in the application — every agency, client, property, unit, tenant, and transaction. This cannot be undone. There is no backup unless you have exported data manually before proceeding.

Only perform a System Reset if you need to completely wipe the system (e.g. for a fresh installation or decommissioning). Do not use this for routine maintenance.

Steps to Perform a System Reset

  1. Ensure you have exported all necessary reports and data using the Print Reports section.

  2. Navigate to "My Profile" in the sidebar.

  3. Scroll down to the System Reset section at the bottom of the profile page.

  4. Enter your Master Admin password to confirm your identity.

  5. Tick the confirmation checkbox: "I understand this will permanently delete ALL data."

  6. Click the "Reset System" button.

  7. A confirmation message appears: "System reset complete. All data cleared. Please refresh the page."

  8. Refresh the browser. The system is now empty and ready for a fresh setup.

Dark / Light Mode

Click the 🌙 moon icon (top right) for dark mode, or ☀️ sun icon to return to light mode. The preference is saved automatically.

Signing Out

Click "Sign Out" at the bottom of the sidebar. You will be returned to the main login page.

Security reminder: Always sign out after each admin session. The Master Admin account has unrestricted system access — treat its credentials with the highest level of confidentiality.
